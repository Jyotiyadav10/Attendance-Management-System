
# Attendance Management System

A full-stack Attendance Management System built with the MERN stack and Next.js, featuring secure role-based authentication (Admin, Teacher, Student), class and subject management, teacher assignments, attendance tracking, reports, and responsive dashboards for efficient academic administration.

## Features

- Admin, Teacher, and Student dashboards
- JWT Authentication
- Attendance management
- Class & Subject management
- Teacher assignment
- Reports & analytics
- Responsive UI

## Tech Stack

- Next.js
- React.js
- Node.js
- Express.js
- MongoDB
- TypeScript

## Installation

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.
